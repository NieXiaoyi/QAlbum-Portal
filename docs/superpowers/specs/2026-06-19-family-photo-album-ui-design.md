# 家庭云相册 UI 设计文档

> QAlbum-Portal — 家庭云相册 Web 应用
> 日期: 2026-06-19
> 状态: 已定稿

---

## 1. 概述

QAlbum-Portal 是一款面向核心小家庭（2-4 人）的家庭云相册 Web 应用，支持桌面端和手机浏览器响应式访问。管理员创建云空间，家庭成员通过注册并经管理员审核后加入，共同管理和浏览家庭照片。

### 核心需求

| 功能 | 说明 |
|------|------|
| 云空间管理 | 管理员创建，成员注册 + 管理员审核加入 |
| 相册管理 | 创建/删除相册，支持列表/平铺双视图切换 |
| 照片上传 | 上传至指定相册，支持多选和拖拽 |
| 照片移动 | 在相册间移动照片 |
| 回收站 | 删除的内容 30 天保留期，可恢复或永久删除 |
| 角色权限 | 管理员（审核成员）+ 普通成员 |

## 2. 视觉风格

**温暖家庭风**

- **主色调：** 米白底色（`#FFFCF8`）、暖色点缀（`#E8C9A0`、`#D4956A`）
- **卡片样式：** 圆角设计（`border-radius: 12px`），柔和阴影
- **字体色彩：** 深棕文字（`#5A4A3A`）、辅助灰（`#B8A090`）
- **整体氛围：** 温馨、亲切、突出照片内容

## 3. 信息架构

### 底部标签栏导航（移动端）/ 顶部导航（桌面端）

```
┌─────────────────────────────────────┐
│         底部 / 顶部 标签栏            │
├──────────┬──────────┬──────┬────────┤
│  📷      │  📂      │ 🗑️   │  👤    │
│  时光轴   │  相册    │ 回收站 │ 设置   │
└──────────┴──────────┴──────┴────────┘
```

四个主要标签页：

1. **时光轴（首页）** — 按时间倒序展示全部照片
2. **相册** — 相册列表 + 相册内照片网格
3. **回收站** — 已删除的照片/相册，可恢复
4. **设置/成员** — 个人设置、成员管理、审核入口

## 4. 页面设计

### 4.1 时光轴首页

**布局：**
- 顶部：云空间名称 + 照片总数 + 搜索/上传按钮
- 主体：照片按年月分组，每组有分隔标题
- 网格：移动端 3 列，桌面端自适应更多列

**交互：**
- 点击照片 → 进入大图查看
- 右上角 + → 上传照片弹窗
- 搜索 → 按照片名或日期搜索

### 4.2 相册页面（双视图模式）

支持**列表模式**和**平铺模式**一键切换，用户偏好本地持久化。

| 模式 | 适用 | 展示方式 |
|------|------|----------|
| ☰ 列表模式 | 移动端（默认） | 一行一个相册，左封面右信息 |
| ▦ 平铺模式 | 桌面端（默认） | 卡片式大图封面，网格排列 |

**相册列表页：**
- 每个相册展示：封面缩略图（照片数量角标）、名称、照片数、更新时间、共同编辑成员
- 右上角 + 号创建新相册

**相册详情页：**
- 顶部：返回、编辑相册、移动照片按钮
- 主体：照片 3 列网格展示
- 点击照片 → 大图查看

### 4.3 回收站

**布局：**
- 顶部标签切换：「照片」/「相册」
- 列表：每项显示缩略图、文件名、来源相册、删除时间、自动清理倒计时
- 每条记录右侧「恢复」按钮

**交互：**
- 30 天保留期，显示自动永久删除日期
- 支持多选批量恢复
- 底部「清空回收站」按钮（需二次确认）

### 4.4 设置 / 成员管理

**分组结构：**

- **云空间信息卡** — 名称、管理员、成员数、照片总数
- **成员管理区：**
  - 👥 所有成员列表
  - 📩 审核申请（管理员可见，红色角标通知）
- **设置区：** 云空间封面、存储用量、个人资料、修改密码
- **退出云空间** — 底部红色警示区域

**审核流程：**

```
用户注册提交申请 → 管理员收到审核通知 → 通过/拒绝 → 结果通知用户
```

### 4.5 照片交互

**大图查看：**
- 深色背景（`#2A2420`）突出照片
- 顶部显示日期，底部文件名 + 来源相册
- 右上角操作：移动、删除
- 左右滑动切换上一张/下一张

**上传流程：**

```
点击 + → 选择照片（多选/拖拽）→ 选择目标相册 → 开始上传
```

**移动流程：**

```
选择照片 → 点击移动 → 选择目标相册（或新建）→ 确认移动
```

**删除流程：**

```
选择照片 → 删除 → 进入回收站 → 30 天内可恢复
```

## 6. 响应式设计策略

| 断点 | 设备 | 布局调整 |
|------|------|----------|
| < 768px | 手机 | 底部标签栏，相册默认列表模式，照片 3 列 |
| 768px - 1200px | 平板 | 顶部导航，相册可切换模式，照片 4-5 列 |
| > 1200px | 桌面 | 顶部导航，相册默认平铺模式，照片 6-8 列 |

视图切换偏好通过 `localStorage` 持久化。

---

## 7. 非功能性设计要点

- **上传区域** 支持点击选择和拖拽上传，上传前显示预览
- **删除确认** 所有删除操作需二次确认对话框
- **清空回收站** 弹出确认对话框，明确提示不可恢复
- **加载状态** 照片网格采用骨架屏占位
- **空状态** 相册为空时展示引导提示
